# Plansio Portfolio Website

## Project Description

This was a university project where I created a portfolio website for a marketing agency called **Plansio**. The website showcases the agency's services and work in a clean, professional manner.

## Overview

This is a simple yet effective portfolio website designed to present Plansio's brand, services, and portfolio to potential clients.

## Technologies Used

- **HTML5** - Structure and markup
- **CSS3** - Styling and layout
- **JavaScript** - Interactivity and dynamic functionality

## Live Demo

Visit the live website here: [Plansio Portfolio Website](https://plansio-porfolio-website.vercel.app/)

## Deployment

The website is deployed and hosted on [Vercel](https://vercel.com/).
